#

### Features:
- Responsive markup
- Error Boundry
- Add, delete, mark as important and done for tasks
- Live search
- Filtering

### Stack:

- HTML
- CSS + SASS ( SCSS )
- ES6+
- React 
- useReducer + Context for global state

#

### Install:

```sh
npm install
```
### Run app:

```sh
npm start
```
